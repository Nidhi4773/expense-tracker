# Expense Tracker 

## 📌 Description
This is a web-based Expense Tracker application that helps users manage their daily income and expenses. It includes features like user login/register, adding transactions, filtering, and tracking balance.

---

## 🚀 Features
- User Authentication (Login & Register)
- Add income and expense transactions
- View total balance, income, and expenses
- Filter transactions by type and date
- Delete transactions
- Dark/Light theme support
- Responsive design

---

## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling & Responsive Design)
- JavaScript (Functionality & Logic)
- LocalStorage (Data storage)

---

## ⚙️ How It Works
- User first registers and logs into the system
- Data is stored in browser using localStorage
- Users can add income or expense entries
- The system automatically calculates balance
- Transactions can be filtered or deleted
- Settings like theme and currency are saved

---

## 💻 Code Overview
- HTML is used to create pages like login, register, and dashboard
- CSS is used for styling, animations, and responsive layout
- JavaScript handles:
  - Form submission
  - Data storage and retrieval
  - Transaction calculations
  - Event handling (clicks, input)
- LocalStorage is used to store user data and transactions
